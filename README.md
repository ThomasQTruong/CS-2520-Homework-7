# Controls
- [RClick] - Shoot.
- [W] / [Up Arrow] - Move up.
- [A] / [Left Arrow] - Move left.
- [S] / [Down Arrow] - Move down.
- [D] / [Right Arrow] - Move right.
- [E] - Next projectile.
- [Q] - Previous projectile.
